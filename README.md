# Rent Ledger

A property management and rent collection app for Kenyan buildings.

## Features

- **Dashboard**: Occupancy tracking, rent collection rates, cash flow analysis
- **Units**: Manage rental units, tenant info, and rent amounts
- **Rent Payments**: Log payments, track arrears, generate receipts
- **Expenses**: Track maintenance, utilities, security, insurance, and other costs
- **Communication**: Send rent reminders via SMS and WhatsApp
- **Print Receipts**: Generate printable/shareable rent receipts

## Tech Stack

- React 18
- Tailwind CSS
- Lucide React icons
- localStorage for data persistence

## Setup

```bash
npm install
npm start
```

## Deploy to GitHub Pages

```bash
npm run deploy
```

Your app will be available at: `https://erastus78.github.io/rent-ledger`

## Local Development

```bash
npm start
```

Runs the app in development mode at [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

Builds the app for production to the `build` folder.
